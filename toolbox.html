前言

Eclipse

Java/Web的开发岂能少了她, 一直都没有换过, 与之能胜任的还有一位[IntelliJ IDEA],听说很厉害,但木有用过,mark先.
一个工具的强大,往往体现在它的插件上, 关于eclipse的插件都是基于在线安装, 点击help->Install New Software. 再点击Add,输入名称,URL; 全选,同意协议, 下一步即可.
官网: http://www.eclipse.org

Eclipse Color Theme

Eclipse颜色主题插件, 可以挑一些自己喜欢的主题样式,如Sublime Text 2.
官网: http://eclipsecolorthemes.org
Name: Theme
Location: http://eclipse-color-theme.github.com/update

SVN

用于项目版本的控制.
官网: http://subclipse.tigris.org
Name: SVN
Location: http://subclipse.tigris.org/update_1.8.x

Emmet

做JSP开发,在网页的编写岂能少了她, 为你可以节约一半的编写代码的时间.
官网: http://emmet.io
Name: Emmet
Location: http://emmet.io/eclipse/updates

Sublime Text 2

对它的功能好处,网上一大把, Google一下,(*^__^*) 嘻嘻
想安装插件,首先安装Package Control,安装过程,代码都在这[https://sublime.wbond.net/installation]
还是先下把, 按Ctrl+~[view-->Hide control]调出控制台，粘贴如下代码到控制台,回车...

import urllib2,os; pf='Package Control.sublime-package'; ipp = sublime.installed_packages_path(); os.makedirs( ipp ) if not os.path.exists(ipp) else None; urllib2.install_opener( urllib2.build_opener( urllib2.ProxyHandler( ))); open( os.path.join( ipp, pf), 'wb' ).write( urllib2.urlopen( 'http://sublime.wbond.net/' +pf.replace( ' ','%20' )).read()); print( 'Please restart Sublime Text to finish installation')
官网: http://www.sublimetext.com

PotPlayer

除了自带的播放器外,这是唯一装在电脑的播放器,没广告,自带解码器,对视频,字幕都表现的很好.
官网: http://cafe.daum.net/pot-tool

Matroska

MKV解码器,应该是播放MKV格式的最好的选择.
